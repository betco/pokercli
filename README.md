Poker CLI
========

This is a CLI client for the pokernetwork. It can be used to play at cashgame tables.

commands
--------

`j` or `j <table_id>` to join a table

`s` to seat

`bi` to buy in


`ch` to check

`f` to fold

`r <amount>` to raise

`c` to call

`le` to leave

`so` to sit out

`si` to sit in

`rebuy <amount` to rebuy

`all_in` to go all in
